Built for the **DIU AI Innovation Hackathon 2026** — a real-time dashboard tracking per-node GPU, CPU, memory, temperature, and running jobs across a simulated cluster.

- Trained an XGBoost scheduler that recommends optimal node placement for incoming jobs against live drifting metrics, designed to swap in real Prometheus/DCGM data with no code changes.

**Tools:** TypeScript, Python, XGBoost, React
